# LiftMark (练刻) 更新日志

## v2.3.0 — 2026-07-02

**小组成员系统重构 + 邀请码 + 训练数据同步**

### 新增功能
- **邀请码系统**：创建小组后可生成邀请码，分享给朋友加入
- **待确认数据**：训练结束后可选择将数据上传给组员，组员登录后确认接受
- **头像文件上传**：头像改为文件上传方式，节省数据库空间
- **成员头像同步**：训练页面自动从服务器同步最新头像

### 服务器端
- 新增 `member_profiles` 表（成员训练资料）
- 新增 `pending_training_data` 表（待确认训练数据）
- 新增 `group_invitations` 表（邀请码）
- 新增邀请码接口：创建、查询、禁用、使用
- 新增待确认数据接口：上传、查询、确认、拒绝
- 新增头像文件上传接口：`POST /auth/avatar/upload`
- Nginx 配置更新：支持静态文件服务

### 手机端
- 新增邀请码管理页面
- 新增加入小组页面（输入邀请码）
- 新增待确认数据通知页面
- 新增训练结束选择上传组员页面
- 头像上传改为 FormData 文件上传方式

## v1.2.0 — 2026-06-22

**训练执行页全面重构 + 多处 UI 优化**

### 训练中页面
- 顶部导航栏改为浅色背景，左侧返回箭头 + 标题"训练中" + 副标题"增力阶段 · 第 3 周 · Day 2"，右侧红色"结束训练"按钮
- 整体从深色主题改为浅色主题，卡片白底、文字深色
- 布局重排：统计条 → 动作 Hero → 成员轮换 → 当前记录 → 休息卡 → 已完成组 → 轮换顺序 → 进度条
- 移除底部重复的"完成本组"按钮，记录卡内已有
- 新增"上一个动作"按钮，支持前后切换动作
- 移除休息卡片中多余的"跳过休息"按钮，只保留记录卡内的动态按钮
- 成员轮换：点击头像可切换当前记录人
- 训练进度条：支持点击任意动作节点跳转

### 记录卡片
- 重量/次数输入与 RPE/RIR 选择器左右分栏显示
- "完成本组"和"跳过休息"按钮并排，完成本组更醒目
- 已完成组支持内联编辑（直接修改重量/次数/RPE/RIR）和删除

### 我的页面
- Banner 背景色从深色改为浅色
- Logo 切换为浅色背景适配版本
- 图标背景色对比度提升

### 记录页面
- 新增"训练分析"快捷入口
- 修复分析页面标题显示问题

### 新增组件
- `ExerciseHeroCard` — 动作 Hero 卡片
- `GroupMemberStrip` — 成员轮换条
- `CurrentSetRecorder` — 当前记录卡片
- `CompletedSetList` — 已完成组列表
- `RotationOrderCard` — 轮换顺序卡片
- `WorkoutProgressStrip` — 训练进度条（支持 card/dock 两种模式）
- `WorkoutStatsBar` — 训练统计条
