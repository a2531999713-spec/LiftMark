# Settings 模块概览

更新时间：2026-06-11

## 1. 模块职责

设置页承接品牌信息、本地小组状态、数据导出入口、计划导入导出入口和开发调试信息。

- 展示 App 名称、版本、当前小组、重量单位和周五补弱状态。
- 提供导出全部数据、导出当前计划、导出训练记录入口。
- 提供导入计划、重置系统方案数据、清空测试数据入口。
- 展示 SQLite 和 seed 初始化状态。

## 2. 非职责

- 不直接实现云同步。
- 不在页面中编写计划导入导出核心逻辑。
- 不绕过确认弹窗执行危险操作。

## 3. 依赖模块

- group
- member
- plan
- export
- history

## 4. 主要文件

| 文件 | 说明 |
|---|---|
| `app/(tabs)/settings.tsx` | 设置页 UI 和入口编排。 |
| `src/services/exportService.ts` | 全量数据和训练记录 JSON 导出、默认 seed 重置。 |
| `src/services/planFileService.ts` | `.liftmark.json` 计划文件生成、校验和导入 ID 重映射。 |

## 5. 修改风险

- 清空数据和重置计划必须二次确认。
- 计划文件不能包含成员 1RM 或训练记录。
- 未实现功能必须显示“开发中”，不能无响应。
