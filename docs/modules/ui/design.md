# UI 模块设计

更新时间：2026-06-14

## 视觉方向

练刻 LiftMark 的界面应表现为成熟的力量训练工具：专业、干净、训练现场友好、数据清楚。整体以浅灰页面背景、白色卡片、深海军蓝训练场景和珊瑚红主行动形成层级。

## 品牌色

- Ink Navy：`#0D1B2A`
- Coral Red：`#FF5A4D`
- Light Gray：`#F2F4F7`
- White：`#FFFFFF`

所有页面通过 `src/theme/` 引用颜色、字号、间距、圆角和阴影，不在页面散落基础视觉参数。

## 页面布局原则

- 探索页强调平台感：搜索、Hero、快捷入口、推荐计划。
- 搭子页强调本地小组和成员管理：成员卡片、训练统计、邀请入口“开发中”。
- 训练页是今日训练驾驶舱：当前计划、状态筛选、成员建议重量、开始训练。
- 计划页是计划管理中心：当前计划、我的计划、内置计划、创建、复制、导入、导出。
- 记录页是数据分析中心：预估 1RM、PR、完成率、训练量、趋势和建议。
- 训练执行页使用沉浸式全屏，不显示底部 Tab。

## 资源设计

- 品牌图标和 wordmark 由 `src/assets/brand/index.ts` 的 `liftmarkBrandAssets` 导出。
- 训练氛围图由 `src/assets/images/index.ts` 的 `liftmarkImages` 导出。
- 页面不得直接引用原始参考图文件名。
- Android 原生图标和启动页资源已同步到 `android/app/src/main/res/`，本地 APK 构建不依赖重新 prebuild。

## 开发中状态

未实现功能必须显示“开发中”并给出反馈，不允许点击无响应。
