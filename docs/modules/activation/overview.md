# Activation 模块概览

更新时间：2026-06-12

## 1. 模块职责

Activation 模块负责第一版本地试用状态和激活码校验。当前不做账号、不做支付、不做微信或手机号登录。

## 2. 第一版能力

- 保存本地试用状态。
- 显示是否已激活、试用剩余天数、设备 ID 和 App 版本。
- 支持本地测试激活码 `LIFTMARK-TEST-2026`。
- 为后续远程校验预留 `RemoteActivationProvider` 接口。

## 3. 边界

- 激活状态存 SQLite。
- 高级导出、计划分享、远程计划库、云同步第一版只提示，不接真实服务。
- 激活建议只影响功能开放提示，不影响本地训练数据完整性。
