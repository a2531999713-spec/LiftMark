# Body Metrics 模块概览

更新时间：2026-06-30

Body Metrics 是训练辅助模块，不是体检或健康诊断模块。它帮助用户快速记录体重、可选记录体脂和围度，并把身体变化和训练频率放在同一视角观察。

## 功能范围

- 快速记录体重、日期和备注。
- 折叠记录体脂、胸围、腰围、臀围、上臂围、大腿围和小腿围。
- 设置增肌 / 减脂 / 维持目标。
- 查看体重、体脂和围度趋势。
- 查看最近变化摘要和近 4 周训练关联摘要。

## 数据边界

- 核心数据写入 SQLite：`body_metrics`、`body_metric_goals`。
- 不使用 AsyncStorage 保存身体记录或目标。
- 不提供医学化诊断文案。
