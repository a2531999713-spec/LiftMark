# Settings 模块概览

更新时间：2026-06-30

## 2026-06-30 账号设置归位

- “我的”页不再直接展示退出登录；退出入口统一放在 `app/account/settings.tsx`。
- 退出登录必须二次确认，并明确不会删除本机训练记录。
- 普通二级页面不再展示默认 Stack 标题，保留简洁返回。

## 2026-07-01 登录入口修正

- 移动端登录页提供双模式：手机号 / 练刻 ID + 密码登录，以及手机号 + 短信验证码登录 / 注册。
- 新手机号可由后端验证码登录接口自动创建账号；服务端生成注册序号、注册时间、来源、活动码和早期用户分层。
- 验证码用于登录、账号创建和后续绑定类敏感能力；UI 不展示服务端 `debugCode`。

## 2026-06-29 登录入口修正

- 旧口径：移动端登录页曾只使用手机号 + 短信验证码登录 / 注册一体入口。
- 旧口径：密码登录 / 密码注册接口曾仅保留在 service 层兼容后端。
- 验证码用于登录、账号创建和后续绑定类敏感能力；UI 不展示服务端 `debugCode`。

## 2026-06-24 登录门禁补充

- 未登录用户不能进入我的页或主 Tab，启动时直接进入账号登录 / 注册页。
- 当前最新移动端入口使用手机号 + 短信验证码登录 / 注册一体流程。
- 我的页账号区现在区分免费版、Pro、永久会员和本机模式。
- 已有本地 session 但服务器不可达时进入本机模式，不主动拉取云端完整资料。
- 账号设置页显示当前账号、安全与权益、云同步和会员与激活；云同步为“开发中 / 可测试”入口。
- 退出登录只清除 session，不清空本机 SQLite 训练数据。

## 2026-06-24 我的页信息架构调整

- `app/(tabs)/settings.tsx` 当前承载“我的”页，而不是旧的调试型设置中心。
- 我的页不显示页面标题，按账号主卡、训练档案 / 小组成员 / 偏好设置 / 账号设置和关于练刻组织；退出登录进入账号设置页。
- 训练数据导出、云同步、会员与激活、账号安全和危险数据操作均进入二级页面；普通我的页不再直接显示重建测试数据、SQLite/seed 状态或重置默认计划。
- 当前 auth service 已接入真实后端验证码发送、验证码登录、密码登录兼容接口、token 保存/刷新和 `/auth/me`；sync 边界复用 `src/sync/*`，不改变本地 SQLite 训练闭环。
- 账号头像与训练成员头像分离；账号头像使用 `account_profile_cache` 和本地文件缓存，成员头像仍归属于 `member_profiles`。

## 2026-06-14 settings-history-plan-switch-cleanup-sprint

- 设置页顶部使用 LiftMark 品牌专属图，展示“练刻 LiftMark / 你的力量训练计划执行器 / 本地数据 / Android 预览版 / com.liftmark.app”。
- 设置页“成员资料”只做入口和摘要，展示全部本地成员，不内嵌 `MemberForm`，不显示“保存成员”或“稍后补充”。
- 设置页“加重单位”是成员个人参数入口，点击后进入全成员加重单位列表，不能默认跳到第一个成员。
- 设置页不再暴露“周五策略”；数据库字段继续保留给训练页兼容，后续策略应放到计划或训练临时覆盖中。
- 设置页不再常驻显示导出 JSON 或“最近导出预览”；导出完成后只给成功提示。
- 设置页和计划页均提供“导入计划”入口，导入 `.liftmark.json` 后写入 SQLite 成为“我的计划”，用户可选择是否设为当前训练计划。

## 1. 模块职责

“我的”页承接账号身份、训练身份、小组、偏好和关于练刻入口；账号设置页承接账号安全、会员与激活和退出登录。

- 展示账号主卡、当前小组、当前计划和训练身份入口。
- 提供成员资料、偏好设置、账号设置和关于练刻入口。
- 普通用户主流程不展示调试型数据维护入口。

## 2. 非职责

- 不直接实现云同步。
- 不在页面中编写计划导入导出核心逻辑。
- 不在设置页直接编辑成员资料。
- 不在设置页暴露周五策略。
- 不在设置页常驻显示 JSON 预览或长文本调试数据。
- 不绕过确认弹窗执行调试保护。

## 3. 依赖模块

- group
- member
- plan
- export
- history

## 4. 主要文件

| 文件 | 说明 |
|---|---|
| `app/(tabs)/settings.tsx` | 设置页 UI 和入口编排。 |
| `app/settings/members.tsx` | 设置页进入的全成员摘要列表。 |
| `app/settings/member-units.tsx` | 设置页进入的全成员加重单位摘要列表。 |
| `src/services/exportService.ts` | 全量数据和训练记录 JSON 导出、默认 seed 重置。 |
| `src/services/planFileService.ts` | `.liftmark.json` 计划文件生成、校验和导入 ID 重映射。 |
| `src/services/planDocumentService.ts` | Expo DocumentPicker 文件选择、读取计划文件并生成导入草稿。 |

## 5. 修改风险

- 计划导出和重置计划必须二次确认。
- 计划文件不能包含成员 1RM 或训练记录。
- 未实现功能必须显示“开发中”，不能无响应。
