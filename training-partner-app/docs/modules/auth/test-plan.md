# Auth 模块测试计划

更新时间：2026-06-24

## 自动化测试

- `src/tests/auth-access.test.ts`
  - 未登录不能使用主应用功能。
  - 登录免费版可以开始训练。
  - 免费版添加第三位成员时返回 Pro 或限制提示。
  - 免费版高级历史返回 Pro 提示。
  - Pro 可以使用 Pro 小组能力。
  - 退出登录策略不清空本地训练数据。
- `src/tests/api-client-errors.test.ts`
  - `fetch failed` / `ConnectException` 映射为“无法连接服务器，请检查网络后重试。”。
  - `Network request failed` 映射为“网络连接失败，请稍后重试。”。
  - Abort / timeout 映射为中文超时提示。

## 手工回归

- 清空 App 数据后首次启动，只显示手机号登录 / 注册页，不能进入首页、计划、记录、我的。
- 登录页不显示继续浏览、预览模式、账号密码登录、账号密码注册、微信登录、Apple 登录、会员权益或小组协作卖点。
- 未勾选协议时，发送验证码和登录 / 注册均提示“请先同意用户协议和隐私政策”。
- 手机号为空或格式错误时不能发送验证码，并显示中文提示。
- 验证码为空或长度不正确时不能登录，并显示中文提示。
- 点击获取验证码进入 loading，成功后提示“验证码已发送”，并进入 60 秒倒计时。
- 发送验证码失败时不显示 `fetch failed`、`ConnectException`、服务器 IP 或 stack trace。
- 登录成功后保存 session 并进入 `/(tabs)/today`。
- 已有 session 时断网重启 App，可以进入本机模式，首页显示“当前离线，已进入本机模式”。
- 离线进入不拉取云端完整资料，不做全量同步，不清空 SQLite。
- 退出登录后本机 SQLite 训练记录仍存在。
