# 推荐计划流程

更新时间：2026-06-30

## 输入

- 训练目标：增肌、增力、减脂保肌、新手入门、恢复训练。
- 每周训练天数：2、3、4、5 或不固定。
- 训练经验：刚开始、3 个月以内、3-12 个月、1 年以上。
- 器械条件：健身房完整器械、家庭基础器械、只有哑铃 / 杠铃、不确定。
- 可选基础力量：深蹲、卧推、硬拉 1RM。

## 输出

`recommendPlans()` 返回按分数排序的推荐计划，第一项为主推荐，其余作为备选展示。

## 当前映射

- 新手或每周 2-3 天：优先新手全身训练计划。
- 增肌且每周 4 天以上：优先上肢 / 下肢分化计划。
- 增力：优先 5x5 基础力量计划。
- 减脂保肌：优先减脂保肌训练计划。
- 恢复训练：优先恢复训练计划。
- 居家或器械有限：优先居家哑铃训练计划。

## 约束

- 推荐结果必须来自系统方案目录。
- 使用推荐计划前必须复制为用户计划。
- 推荐逻辑不依赖 UI 状态和远程服务。
