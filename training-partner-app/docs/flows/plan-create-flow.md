# 创建计划流程

更新时间：2026-06-12

## 1. 目标

用户可以从空白创建自己的训练计划。第一版先保留入口，深层编辑器后续 Sprint 接入。

## 2. 目标字段

- 计划名称。
- 训练目标。
- 每周训练频率。
- 周期周数。
- 训练日。
- 动作。
- 组数。
- 次数。
- 百分比。
- RPE / RIR。
- A/B/C 优先级。
- 休息时间。

## 3. 计划来源

空白创建的用户计划建议使用：

```text
source = blank_created
originSchemeId = null
```

## 4. 当前实现

计划页已提供“创建计划”入口。深层编辑能力暂未实现，点击显示“开发中”。

## 5. 后续实现要求

- 创建后的计划必须进入“我的计划”。
- 创建后的计划可以设为当前计划。
- 创建计划不能修改系统方案。
- 创建计划不能覆盖已有计划。
