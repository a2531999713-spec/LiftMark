# 版本更新记录

本文档详细记录 **练刻 LiftMark** 每个版本的更新内容，方便用户快速了解每个版本的新功能、修复和优化。

---

## v1.2.0 — 2026-06-30

### 🎯 核心更新

#### 训练执行页全面重构
- 顶部导航栏改为浅色背景，显示训练标题和进度
- 整体从深色主题改为浅色主题，提升可读性
- 布局重排：统计条 → 动作 Hero → 成员轮换 → 当前记录 → 休息卡 → 已完成组 → 轮换顺序 → 进度条
- 新增"上一个动作"按钮，支持前后切换动作
- 成员轮换：点击头像可切换当前记录人
- 训练进度条：支持点击任意动作节点跳转

#### 记录卡片优化
- 重量/次数输入与 RPE/RIR 选择器左右分栏显示
- "完成本组"和"跳过休息"按钮并排，完成本组更醒目
- 已完成组支持内联编辑（直接修改重量/次数/RPE/RIR）和删除

#### 我的页面更新
- Banner 背景色从深色改为浅色
- Logo 切换为浅色背景适配版本
- 图标背景色对比度提升

### 🆕 新增组件
- `ExerciseHeroCard` — 动作 Hero 卡片
- `GroupMemberStrip` — 成员轮换条
- `CurrentSetRecorder` — 当前记录卡片
- `CompletedSetList` — 已完成组列表
- `RotationOrderCard` — 轮换顺序卡片
- `WorkoutProgressStrip` — 训练进度条（支持 card/dock 两种模式）
- `WorkoutStatsBar` — 训练统计条

### 🐛 修复
- 修复分析页面标题显示问题

### 📝 文档
- 更新训练执行页相关文档

## v1.1.0 — 2026-06-17 ~ 2026-06-22

### 🎯 核心更新

#### UI 全面重新设计
- 4Tab 布局重构（首页/计划/记录/我的）
- 主题系统全面优化（颜色、圆角、阴影）
- 应用 taste-skill / impeccable 设计规范

#### 新增功能
- 新增 MiniLineChart 迷你折线图组件
- 首页仪表盘重设计
- 训练记录和分析功能增强

### 🐛 修复
- 多处 UI 交互问题修复
- 性能优化和稳定性提升

## v1.0.0 — 2026-06-14 ~ 2026-06-15

### 🎯 核心功能

#### 项目初始化
- 完整训练伙伴应用源码
- 技术栈：React Native + Expo + SQLite

#### 功能模块
- **训练记录**：详细记录每次训练的组数、次数、重量
- **计划管理**：系统化的力量训练和增肌计划
- **历史分析**：查看训练历史和进步趋势
- **运动库**：100+ 个系统动作
- **成员管理**：支持多个团队成员
- **设置**：单位、记录方式、休息计时

## 升级建议

### 从 v1.0.x 升级到 v1.1.0
- 享受全新的 UI 设计和更好的用户体验
- 新增迷你折线图，数据可视化更清晰
- 建议备份重要数据后升级

- 体验重构后的训练执行页，操作更流畅
- 浅色主题更适合日常使用
